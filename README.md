# 📊 Agri Market Intelligence & Price Forecasting Dashboard

## 🚀 Overview

This project is an end-to-end Agri Market Intelligence solution developed using **Power BI**, **SQL**, and **Excel** to analyze agricultural commodity prices, MSP (Minimum Support Price), market arrivals, seasonal trends, and short-term price forecasts.

The dashboard helps identify commodities trading below MSP, analyze procurement risks, monitor arrivals across markets, and visualize 30-day price forecasts with confidence intervals.

---

## 🎯 Business Problem

Farmers and policymakers often lack a single platform to monitor:

- Market prices across commodities
- MSP comparison
- Procurement risks
- Arrival trends
- Seasonal price behaviour
- Short-term price forecasts

This project addresses these challenges through an interactive Power BI dashboard.
